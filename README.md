## Project Objective
This project aims to analyze customer data and present key insights through an interactive Excel dashboard. The dashboard is designed to help stakeholders understand customer growth trends, customer status, demographic distribution, membership levels, social media performance, and geographic distribution, supporting data driven decision making for marketing and customer management strategies.
## Dataset used
- <a href="https://github.com/tifazulfa/Data-Analysis-Dashboard/blob/main/Data%20Customer%20Analysis.xlsb">Dataset</a>
## Key Business Questions (KPIs)
- How does the monthly customer trend change over time?
- Which month has the highest and lowest number of customers?
- What is the distribution of customer status (Active, Non-Active, Pending)?
- How are customers distributed by gender?
- What is the membership distribution (Platinum, Gold, Silver, Bronze)?
- Which social media platforms contribute the most customers?
- How are customers geographically distributed across regions?
